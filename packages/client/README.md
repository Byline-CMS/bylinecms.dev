# @byline/client

In-process SDK for reading and writing Byline documents — a higher-level,
DSL-like query and mutation API built over the storage primitives and the
`document-lifecycle` service. Provides `find`, `findOne`, `findById`,
`findByPath`, `count`, `create`, `update`, `delete`, `changeStatus`, with
field-level filters, sorting, relationship population, status-aware reads,
and the `beforeRead` / `afterRead` collection hooks.

This package is part of [Byline CMS](https://github.com/Byline-CMS/bylinecms.dev)
— a developer-friendly, open-source headless CMS with versioning, editorial
workflow, and content translation as first-class concerns.

For documentation, the full architecture overview, and getting started
instructions, see the main repository:
<https://github.com/Byline-CMS/bylinecms.dev>.
